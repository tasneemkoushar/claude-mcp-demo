Claude MCP Demo

A demonstration of Claude integrated with GitHub and Jira via the Model Context Protocol.

View on GitHub

Features

📉

GitHub MCP

Manage repositories, pull requests, and issues directly through Claude using the GitHub MCP server.

📋

Jira MCP

Read and interact with Jira tickets seamlessly from within your Claude workflow.

⚙

GitHub Actions

Automated deployment to GitHub Pages via GitHub Actions on every push to main.

About

This project showcases the power of the Model Context Protocol (MCP) — enabling AI assistants like Claude to interact with external tools and services directly.
